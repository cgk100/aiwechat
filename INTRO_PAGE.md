# H5 系统介绍页面

## 📱 访问地址

- **开发环境**: http://localhost:3000/intro
- **首页**: 访问 http://localhost:3000 会自动跳转到介绍页面

## ✨ 页面特性

### 1. **现代化设计**
- 🎨 渐变色主题，视觉冲击力强
- 📱 完全响应式设计，支持移动端/平板/桌面端
- ⚡ 流畅的动画和过渡效果
- 🎯 清晰的信息层级

### 2. **内容板块**

#### Hero 区域
- 醒目的标题和口号
- 简洁的产品介绍
- 明确的 CTA 按钮（进入系统、了解更多）

#### 数据统计
- 4 个核心数据指标展示
- 图标化的视觉表达
- Hover 动画效果

#### 核心功能展示
- 4 大功能模块介绍：
  - 👥 好友管理
  - 🤖 AI 智能答复
  - 📨 营销推广
  - ⚙️ 系统配置
- 交互式切换展示
- 详细功能点列表

#### 技术架构
- 6 项核心技术栈展示
- 图标化的技术说明
- 技术优势介绍

#### 应用场景
- 3 种典型应用场景：
  - 🛍️ 电商客服
  - 🏢 企业服务
  - 📚 教育培训

#### CTA 区域
- 醒目的行动号召
- 渐变背景强调

#### 页脚
- 快速导航链接
- 联系方式
- 版权信息

### 3. **交互特性**

- **动态功能切换**: 点击不同功能模块，右侧展示对应的详细信息
- **悬停效果**: 所有卡片和按钮都有 hover 动画
- **平滑滚动**: 锚点跳转带有平滑滚动效果
- **响应式导航**: 固定顶部导航栏，支持快速跳转

### 4. **视觉设计**

#### 颜色系统
- **主色调**: 青色到绿色的渐变（primary → secondary）
- **辅助色**: 多彩的功能卡片渐变
- **背景**: 灰色渐变柔和背景
- **文字**: 清晰的文字层级（标题/正文/辅助）

#### 字体规范
- **大标题**: 48px - 60px (text-5xl - text-6xl)
- **中标题**: 32px - 40px (text-4xl)
- **小标题**: 20px - 24px (text-xl - text-2xl)
- **正文**: 16px - 18px (text-base - text-lg)

#### 间距规范
- **区块间距**: py-20 (80px)
- **卡片间距**: gap-8 (32px)
- **元素间距**: gap-4 (16px)

## 🎯 使用场景

### 1. **首页展示**
- 作为系统的首页，向新用户介绍产品
- 系统默认首页路由已设置为 `/intro`

### 2. **产品宣传**
- 可以作为独立的产品介绍页面
- 适合分享给潜在用户

### 3. **功能导航**
- 通过页面可以快速跳转到各个功能模块
- 顶部导航栏提供完整的功能入口

## 📱 响应式设计

### 断点说明
- **移动端** (< 768px): 单列布局
- **平板** (768px - 1024px): 双列布局
- **桌面端** (> 1024px): 三列布局

### 移动端优化
- 导航菜单自动调整
- 卡片堆叠显示
- 字体大小自适应
- 触摸友好的交互

## 🚀 性能优化

1. **图片优化**: 使用 emoji 图标，无需加载外部图片
2. **动画性能**: 使用 CSS transform 和 opacity
3. **代码分割**: Next.js 自动代码分割
4. **懒加载**: 组件按需加载
